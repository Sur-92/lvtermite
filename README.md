# Lebanon Valley Termite &amp; Pest Control Co., Inc.

Single-page website for Lebanon Valley Termite &amp; Pest Control — locally owned in Lebanon County, PA since 1994.

**Live preview:** https://sur-92.github.io/lvtermite/

## Structure
- `index.html` — the entire site (inline CSS + JS, no build step, no dependencies)
- `logo.jpg` — company logo
- `img/` — photography (licensed via Pexels, commercial use, no attribution required)

## Editing
It's one self-contained file. Open `index.html` in any browser to preview locally, or edit and re-commit — GitHub Pages redeploys automatically.

Intended production domain: **lvtermite.com**
